# 📞 Phone Directory System

## 📌 Overview
A Python console application that simulates a phone directory.
Users can search for contacts by phone number or name, and add 
new contacts to the directory — built using dictionaries, loops, 
and conditional logic.

## ⚡ Features
- Search contact by phone number
- Search contact by name
- Add a new contact to the directory
- Input validation (rejects invalid phone numbers)
- Continuous interactive menu (runs until user exits)

## 🛠️ Skills Demonstrated
- Python Dictionaries (key-value data storage)
- Loops (`while`, `for`)
- Conditional Statements (`if`, `elif`, `else`)
- Input Validation (`isdigit()`, `len()`)
- Nested Logic

## ▶️ How to Run
```bash
python phone_directory.py
```
